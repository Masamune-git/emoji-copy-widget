## emoji-copy-widget
emoji-copy-widget is a widget tool that allows you to copy the displayed Emoji to the clipboard by clicking on them.


## Installation

```
$ npm install emoji-copy-widget
```

## Usage

 - Launch the emoji-copy-widget
```
$ emoji-copy-widget
```
